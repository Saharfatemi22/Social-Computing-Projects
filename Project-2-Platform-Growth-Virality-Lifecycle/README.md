# Project 2 – Platform Growth, Virality, and User Interactions

This project is part of the **Social Computing** course at the **University of Oulu**.

The goal of this project is to analyze platform growth, engagement, and user interaction patterns using **Python**, **SQL**, and an SQLite database.

---

## Tasks

### 2.1 – Growth
Estimate future server requirements based on historical platform growth, assuming three years of operation and 20% redundancy.

### 2.2 – Virality
Identify the three most viral posts in the history of the platform and justify the metric used to define virality.

### 2.3 – Content Lifecycle
Analyze how long it takes for posts to receive their first and last engagement.

### 2.4 – Connections
Identify the top three user pairs with the strongest mutual engagement and describe the engagement metric.

---

## Files

- `project2_analysis.py` – Python code for Tasks 2.1–2.4  
- `database.sqlite` – SQLite database used for analysis
